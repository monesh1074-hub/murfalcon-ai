# 🚀 Project Name
MURF-FALCON-INTERVIEWER

## 🎯 Problem Statement
Mock interviews are expensive
YouTube videos can’t give real-time feedback
Friends don’t give honest technical feedback
Most people freeze when they have to speak

Result:
Students lose opportunities. Companies get wrong candidates.
We wanted to solve this with real voice conversation, not just text chat."

## 💡 Solution
"So we built Murf Falcon — a full voice-first AI interviewer that feels exactly like talking to a real HR manager.
Features:

You speak → it listens (speech-to-text)
Falcon replies with real Murf AI voice (Natalie — super natural)
Live transcript on the side
Smart follow-up questions using Gemini AI
Instant scorecard: Confidence, Clarity, Technical depth
Supports English + Hindi mixing
Fully working login, history, and database"

## ⚙️ Tech Stack
"Built in just a few days using:
React + Tailwind 
Node.js + Express + MySQL backend
Murf AI for ultra-realistic voice 
Google Gemini for intelligent replies
Full authentication, session management & voice logging

## 🧠 Future Improvements
1.Voice Cloning (Your Own Voice)
Use Murf’s Voice Cloning API — users upload 2–10 minutes of their voice and the interviewer speaks in their own voice.

2.Company-Specific Interview Modes
Add “Google”, “Amazon”, “TCS”, “Startup” presets with custom question banks + difficulty.

3.Multi-Language Full Support
Full Hindi + English + Tamil + regional languages using Murf’s 35+ language support.

4.Video Interview Mode
Add webcam + screen sharing so candidates can show code while speaking (using WebRTC).

5.Detailed Analytics Dashboard
After interview:
Speaking speed graph
Filler words count (“um”, “like”)
Strongest/weakest answers
Improvement tips
